# SMART Constrained Results

This folder stores constrained-variant experiment artifacts.

Canonical file:
- `smart_constrained_v0_metrics.json`: constrained variant selection snapshot with baseline reference and variant grid metadata.
- `smart_checkpoint_eval_v0.json`: checkpoint-level eval planning/runs and metric ingestion snapshot.
- `smart_comparative_v0_report.json`: cross-model comparison report with feasibility and selected candidate.
